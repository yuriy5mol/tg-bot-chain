# 🤖 TG Bot Chain — Генератор Telegram-ботов

Автоматическая генерация Telegram-ботов из текстового описания с помощью LangChain.

## Как это работает

Скрипт использует цепочку из **3 звеньев LangChain**:

| # | Звено | Описание |
|---|-------|----------|
| 1 | **analysis_chain** | Анализирует текстовое задание → выделяет команды, обработчики, формат ответов, зависимости, необходимость БД |
| 2 | **code_chain** | На основе анализа генерирует полный рабочий код бота на aiogram 3.x |
| 3 | **review_chain** | Проверяет код: синтаксис, импорты, структуру запуска. Исправляет ошибки, если находит |

На выходе — готовый `.py` файл, который можно запускать без модификаций.

## Требования

- Python 3.10+
- API-ключ OpenAI (или совместимого провайдера)

## Установка

```bash
# Клонировать / перейти в директорию
cd tg-bot-chain

# Создать виртуальное окружение
python -m venv .venv

# Активировать (Windows PowerShell)
.\.venv\Scripts\Activate.ps1

# Активировать (Linux / macOS)
source .venv/bin/activate

# Установить зависимости
pip install -r requirements.txt
```

## Настройка

Установите переменные окружения:

```bash
# Обязательно — ключ API
set OPENAI_API_KEY=sk-...          # Windows
export OPENAI_API_KEY=sk-...       # Linux/macOS

# Опционально — модель (по умолчанию gpt-4o)
set MODEL_NAME=gpt-4o
```

Или создайте файл `.env` по образцу `.env.example`.

## Использование

```bash
python script.py "Описание вашего бота"
```

### Примеры

```bash
# Бот с мемами
python script.py "Бот, который отправляет случайные мемы"

# Бот-напоминалка
python script.py "Бот для создания напоминаний с указанием времени"

# Бот-переводчик
python script.py "Бот, который переводит текст с русского на английский и обратно"

# Бот-опросник
python script.py "Бот для проведения опросов с инлайн-кнопками и подсчётом голосов"
```

## Результат

После выполнения скрипт создаёт файл `generated_bot.py` в текущей директории.

Для запуска сгенерированного бота:

```bash
set BOT_TOKEN=123456:ABC-DEF...    # ваш токен от @BotFather
python generated_bot.py
```

## Стек технологий

- **LangChain** — оркестрация LLM-цепочки
- **OpenAI API** (gpt-4o по умолчанию) — генерация и анализ
- **aiogram 3.x** — целевой фреймворк генерируемых ботов
- **ast** — финальная валидация синтаксиса Python-кода

## Совместимость с другими LLM

Скрипт использует `ChatOpenAI`, который совместим с любым OpenAI-совместимым API.
Для использования альтернативного провайдера установите `OPENAI_API_BASE`:

```bash
set OPENAI_API_BASE=http://localhost:1234/v1   # например, LM Studio
set MODEL_NAME=llama-3
```
